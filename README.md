# Digital Tools Buying Website

A responsive React application where users can browse digital tools and manage a cart system.

## Technologies

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON

## Features

- Product listing from JSON
- Add to cart and remove functionality
- Cart total calculation
- Toggle between product and cart
- Toast notifications
- Fully responsive design
